TREE
----------

TREE IDA Pro plug-in for Windows XP/7. 


Features
-----------

1.) TREE Dcumentation is located in C:\Users\Current_User_Name\Documents\TREE\documentation\.
Key documents include "TREE User Manual.pdf", "TREE Design Manual.pdf" and "license.rtf"

1.) The Test Suite is located in C:\Users\Current_User_Name\Documents\TREE\Taint Analyzer Test Suite\
The Test Suite contains sample test programs for the TREE IDA Pro plugin.

2.) Please read the TREE manual in C:\Users\Current_User_Name\Documents\TREE\documentation\ for instructions on how to run the sample programs.

License
-----------
See License.rtf file, located in TREE Documentation folder, for TREE license terms.
